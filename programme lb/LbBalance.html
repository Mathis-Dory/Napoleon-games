Bienvenue sur GDE linebattle balance

Ce programme est developpé par Cody de la GDE. Son utilité est très simple, il permet de créer des équipes équilibrées pour les linebattles sur Holdfast.

Fonctionnement

  1. Cliquez sur ajouter un régiment.
  2. Entrez le nom du régiment.
  3. Entrez le nombre de participants.
  4. Repondez aux questions qui apparaitront.
  5. Repétez l'opération pour tous les régiments.
  6. Cliquez lancer la balance.
  7. Reload la page si vous avez fait une erreur puis recommencez.
  8. Entrez le nom des équipes.
  9. Changez le couleur du tableau.
